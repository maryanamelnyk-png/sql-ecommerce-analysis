# SQL Final Project — E-commerce Analysis

## Project Overview
This project focuses on analyzing e-commerce data using SQL to extract key business insights.  
The goal is to answer real-world analytical questions related to sales performance, customer behavior, and order quality.

## Objectives
- Write SQL queries to analyze business metrics  
- Use aggregation functions (SUM, COUNT, AVG)  
- Apply GROUP BY and HAVING  
- Use subqueries for deeper analysis  
- Provide explanations for each query  

## Dataset Description
The dataset contains e-commerce transaction data, including:
- user_id  
- country  
- category  
- total_edited (order value)  
- delivery_status  

## Key Business Questions

### Average Order Value by Category
Analyzes which product categories generate higher revenue per order.

### Total Sales by Country
Identifies top-performing countries based on total sales.

### Returns & Cancellations by Country
Evaluates delivery quality and customer satisfaction.

### Total Cancelled Orders
Provides an overview of problematic orders.

### High-Value Customers
Identifies users with above-average spending using a subquery.

## Skills Demonstrated
- SQL (SELECT, WHERE, GROUP BY, HAVING)  
- Aggregations (SUM, AVG, COUNT)  
- Subqueries  
- Analytical thinking  


## Conclusion
This project demonstrates the ability to analyze business data using SQL and extract actionable insights such as:
- revenue distribution  
- customer segmentation  
- order quality issues  

These insights can support business decisions in marketing, operations, and product strategy.
